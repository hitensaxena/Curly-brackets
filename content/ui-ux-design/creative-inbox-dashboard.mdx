## Challenge

Creative tools often collapse because the inbox becomes a graveyard for
uncategorized inputs.

## Direction

The dashboard separated active work, references, and low-priority captures while
keeping all three visible in the same system.

## Outcome

That separation made the dashboard feel less like storage and more like a place
to begin.
